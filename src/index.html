Frequently Asked Questions

Everything you need to know about using the PickupPal WhatsApp bot for your pickup games.

How do I add the bot to my WhatsApp group?

Contact us to get the bot added to your group!

Is PickupPal free to use?

Yes! PickupPal is free to use for basic features. We offer a premium tier with advanced features like attendance tracking, automated reminders, and custom settings for a small monthly fee. The free tier has everything you need to organize great pickup games.

How many players can join a game?

You can set any number of player slots when creating a game. For example: !!creategame 2024-06-15 7:00 PM 12 Downtown Field creates a game with 12 player slots. There's no upper limit, so you can organize anything from tennis matches to full soccer teams.

Can I schedule recurring games?

Yes! Use the !!createrecurring command to set up weekly or monthly games. For example: !!createrecurring weekly thursday 7:00 PM 10 LA Fitness will create a game every Thursday at the same time and location.

How do reminders work?

PickupPal automatically sends reminders to all signed-up players 24 hours and 2 hours before the game. Premium users can customize reminder timing and add additional reminders if needed.

Does it work in all countries?

Yes! Since PickupPal works through WhatsApp, it's available anywhere WhatsApp operates. The bot supports English language commands, with more languages coming soon.
